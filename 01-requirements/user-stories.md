## Epic 14: Dashboard 与鉴权

### US-14.1 登录页面
**作为** 管理员，
**我想** 在浏览器打开路由器 IP 后看到登录页面，用系统账号密码登录，
**以便** 只有授权用户能管理路由器。

**验收标准**：
- [ ] 访问 http://192.168.21.1 重定向到 https:// 登录页
- [ ] 使用系统用户（root/sudo 用户）账号密码登录
- [ ] 登录失败后有明确提示
- [ ] 连续 5 次失败锁定 15 分钟
- [ ] 登录成功后跳转到 Dashboard
- [ ] 30 分钟无操作自动登出
- [ ] Session 显示在页面右上角，可手动登出

### US-14.2 Dashboard - 网络与系统运行状态
**作为** 管理员，
**我想** 登录 Dashboard 后第一眼看到网络拓扑图和所有代理通道的状态，
**以便** 快速判断网络是否正常。

**验收标准**：
- [ ] 页面上方显示图形化网络拓扑图
- [ ] WAN/LAN 接口关系清晰可见
- [ ] Tailscale Exit Node 状态（在线/离线/延迟）
- [ ] OpenClash 节点状态（延迟/在线/切换）
- [ ] WireGuard 隧道状态（对端在线/传输量）
- [ ] 实时上下行速率仪表（折线图，自动刷新）
- [ ] 地图组件显示各链路的网络地理位置
- [ ] conntrack 连接数
