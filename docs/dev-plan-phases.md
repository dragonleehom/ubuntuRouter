## 三、验证方案

每完成一个 Phase，需验证：

1. **功能测试**: 后端 API 正常工作（curl/浏览器）
2. **UI 测试**: 前端页面无报错，表单提交正常
3. **系统影响**: 不破坏已有功能（接口/防火墙/VPN等）
4. **部署验证**: `ubunturouter.sh update` 后全新安装可用
